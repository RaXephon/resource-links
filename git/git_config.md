#### Git config

```
$ git config -l
git config --global user.name "resh sha"

$ git config -l
user.name=resh sha

$ git config --global user.email "reshama@somemail.com"

$ git config --global core.editor emacs         # to set editor in general, Editor shell

$ git config -l                                 # list git configuration parameters


# to ignore some files, to tell git to ignore README.md~
$ ls -a ~
$ ls -a ~/.git*
```
